## Backend

- `GET		/api/films`
- `GET		/api/films/:id`
- `GET		/api/categories`
- `GET		/api/categories/:id`
- `POST		/api/films`
- `POST		/api/categories`
- `PUT		/api/films`
- `DELETE	/api/films`
- `GET		/api/films-categories`

All resources return JSON.
